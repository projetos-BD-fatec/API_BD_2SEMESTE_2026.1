# 📘 User Story - US06

Como professor, desejo visualizar as disciplinas que já organizei para escolher aquela que desejo planejar.

<br>

## 📌 Explicação

Essa funcionalidade permite que o professor visualize todas as disciplinas já cadastradas para acessar e continuar o planejamento de uma delas.<br>
O objetivo é facilitar a navegação entre disciplinas, permitindo que o professor escolha rapidamente qual deseja planejar ou revisar.<br>
<br>

## 📌 Informações exibidas

Para cada disciplina, devem ser apresentados:<br>

- Nome da disciplina<br>
- Curso<br>
- Semestre<br>
<br>

## 📌 Regras de Negócio
- Devem ser exibidas todas as disciplinas cadastradas<br>
- Cada disciplina deve ser selecionável<br>
<br>
