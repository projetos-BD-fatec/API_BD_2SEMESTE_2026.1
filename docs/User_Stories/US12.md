# 📘 User Story - US12

Como professor, desejo ter meu próprio cadastro para ter acesso apenas aos planejamentos de aulas das minhas disciplinas.
<br> 

## 📌 Explicação
Essa funcionalidade permite que cada professor possua um cadastro próprio, garantindo que ele visualize e gerencie apenas os planejamentos das disciplinas sob sua responsabilidade.<br>
O objetivo é organizar o acesso às informações, evitando que um professor visualize ou altere planejamentos de outros professores.<br>
<br> 

## 📌 Informações exibidas
Devem ser apresentados no cadastro:<br>
- Nome do professor<br>
- Identificador de acesso (e-mail e CPF)<br> 
<br> 

## 📌 Regras de Negócio
- Cada professor deve possuir um cadastro único<br>
- Um professor só pode visualizar e acessar suas próprias disciplinas<br>
- Cada disciplina deve estar vinculada a um professor<br>
- Um professor pode possuir uma ou mais disciplinas<br>
- O acesso às informações deve ser controlado com base no cadastro do professor<br> 
<br> 

## 📌 Critérios de Aceitação
- Deve ser possível cadastrar um professor<br>
- O professor deve acessar apenas suas disciplinas<br>
- As disciplinas exibidas devem estar corretamente associadas ao professor<br>
- Não deve ser possível visualizar ou acessar dados de outro professor<br>
- As informações devem ser consistentes com o cadastro realizado<br> 
<br>
