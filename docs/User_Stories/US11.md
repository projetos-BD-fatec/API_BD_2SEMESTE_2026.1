# 📘 User Story - US11

Como professor, desejo cadastrar uma disciplina informando nome, curso, semestre e carga horária para iniciar o planejamento das aulas do semestre.
<br>

## 📌 Explicação

Essa funcionalidade permite que o professor registre uma nova disciplina, que servirá como base para o planejamento das aulas.<br>
O objetivo é estruturar o contexto necessário para organizar o semestre.<br>

<br>

## 📌 Informações exibidas

Devem ser informados:<br>

- Nome da disciplina<br>
- Curso<br>
- Semestre<br>
- Carga horária<br>
<br>

## 📌 Regras de Negócio
- Todos os campos devem ser obrigatórios<br>
- A carga horária deve ser um valor válido (40 ou 80 horas)<br>
<br>
