## 📘 User Story - US18
**Como professor, quero que meu conteúdo de aulas apareça no meu calendário do celular para eu conseguir me programar antecipadamente para as aulas.**  

<br>

## 📌 Explicação  
Essa funcionalidade permite que o planejamento das aulas seja sincronizado com o calendário do celular do professor.<br>  
O objetivo é facilitar o acompanhamento das aulas e permitir melhor organização da rotina.<br>  

<br>

## 📌 Informações exibidas  
Cada evento deve apresentar:<br>  

- Nome da disciplina<br>  
- Tópico da aula<br>  
- Data<br>  
- Horário<br>  

<br>

## 📌 Regras de Negócio  
- Apenas aulas planejadas devem aparecer no calendário<br>  
- Alterações no planejamento devem atualizar os eventos<br>  
- As informações exibidas devem refletir o planejamento atual<br>  

<br>

## 📌 Critérios de Aceitação  
- Deve ser possível visualizar as aulas no calendário do celular<br>  
- Cada evento deve conter disciplina, tópico, data e horário<br>  
- Alterações no planejamento devem atualizar o calendário<br>  

<br>
