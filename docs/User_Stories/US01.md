# 📘 User Story - US01

Como professor, desejo visualizar os dias da semana e horários das aulas cadastrados para ter fácil acesso a essas informações.

<br>

## 📌 Explicação

Essa funcionalidade permite que o professor visualize, de forma clara e organizada, todos os dias da semana e horários já definidos para uma disciplina.<br>
O objetivo é facilitar a consulta dessas informações, permitindo uma melhor compreensão da distribuição das aulas ao longo da semana.<br>

<br>

## 📌 Informações exibidas

Para cada registro, devem ser apresentados:<br>

- Dia da semana (ex: Segunda, Terça)<br>
- Horário de início e fim (ex: 18:45 - 20:25)<br>
- Quantidade de aulas no dia (ex: 2 aulas)<br>
<br>

## 📌 Regras de Negócio

Devem ser exibidos todos os horários cadastrados da disciplina selecionada<br>
Uma disciplina pode possuir mais de um dia/horário cadastrado<br>
A quantidade de aulas deve ser coerente com o intervalo de horário informado<br>
Caso não existam horários cadastrados, deve ser exibida uma mensagem informativa<br>
<br>

## 📌 Critérios de Aceitação
- Ao acessar a disciplina, os horários cadastrados devem ser exibidos<br>
- Cada registro deve conter dia, horário e quantidade de aulas<br>
- Os dados devem ser apresentados de forma organizada (ex: tabela ou lista)<br>
<br>

## 📌 Exemplo
| Dia     | Horário       | Aulas |
| ------- | ------------- | ----- |
| Segunda | 18:45 - 19:35 | 1     |
| Quarta  | 18:45 - 20:25 | 2     |
