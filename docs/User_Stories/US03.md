# 📘 User Story - US03

Como professor, desejo que minhas aulas sejam distribuídas pelos dias letivos que constam no calendário acadêmico para que eu não precise cadastrar os dias de aula manualmente.

<br>

## 📌 Explicação

Essa funcionalidade é responsável por gerar automaticamente todas as aulas do semestre com base no calendário acadêmico e nos dias/horários definidos para a disciplina.<br>
O professor não deve precisar cadastrar manualmente cada data de aula, pois essa distribuição deve ser feita automaticamente considerando os dias da semana em que a disciplina ocorre.<br>
O objetivo é automatizar a criação do cronograma de aulas, reduzindo esforço manual e evitando erros na definição das datas.<br>

<br>

## 📌 Informações exibidas

Para cada aula gerada, devem ser apresentados:<br>

- Data da aula (ex: 02/03/2026)<br>
- Dia da semana (ex: Segunda)<br>
- Horário da aula (ex: 18:45 - 19:35)<br>
- Quantidade de aulas no dia (quando aplicável)<br>
<br>

## 📌 Regras de Negócio
- As aulas devem estar cadastradas no sistema, não é o professor que deverá cadastrá-las<br>
- As aulas devem considerar o período do semestre<br>
- Para cada dia da semana configurado, devem ter aulas recorrentes ao longo de todo semestre<br>
- A quantidade de aulas por dia deve respeitar o horário informado<br>
<br>

## 📌 Critérios de Aceitação
- Cada aula deve possuir data, dia da semana e horário corretamente definidos<br>
- As aulas devem cobrir todo o período do semestre<br>
- As datas de aula não podem ser inseridas manualmente pelo professor<br>
<br>

## 📌 Exemplo
| Data       | Dia da semana     | Horário       |
