# 📘 User Story - US10

Como professor, desejo exportar o planejamento de aulas para facilitar subir em outras plataformas.

<br>

## 📌 Explicação

Essa funcionalidade permite que o professor exporte o planejamento de aulas já organizado para uso externo.<br>
O objetivo é facilitar o envio e utilização das informações em outros sistemas institucionais.<br>

<br>

## 📌 Informações exibidas

O arquivo exportado deve conter:<br>

- Datas das aulas<br>
- Horários<br>
- Tópicos planejados<br>
<br>

## 📌 Regras de Negócio
- O planejamento deve ser exportado no formato estruturado (ex: planilha)<br>
- Todas as informações relevantes devem ser incluídas<br>
- O arquivo deve refletir o planejamento atual<br>
<br>
