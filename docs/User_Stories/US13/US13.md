# 📘 User Story - US13  
**Como professor, desejo conseguir selecionar uma das disciplinas que eu leciono para realizar o planejamento das aulas.**  

<br>

## 📌 Explicação  
Essa funcionalidade permite que o professor escolha uma disciplina específica para realizar ou continuar o planejamento das aulas.<br>  
O objetivo é garantir que o planejamento seja feito na disciplina correta, facilitando a organização do semestre.<br>  

<br>

## 📌 Informações exibidas  
Devem ser apresentados:<br>  

- Nome da disciplina<br>  
- Curso<br>  
- Semestre<br>  

<br>

## 📌 Regras de Negócio  
- O professor deve visualizar apenas disciplinas vinculadas ao seu cadastro<br>  
- Apenas uma disciplina pode ser selecionada por vez<br>  
- Ao selecionar uma disciplina, o planejamento correspondente deve ser carregado<br>  

<br>

## 📌 Critérios de Aceitação  
- Deve ser possível visualizar as disciplinas cadastradas<br>  
- Deve ser possível selecionar uma disciplina<br>  
- O planejamento da disciplina selecionada deve ser exibido corretamente<br>  

<br>
