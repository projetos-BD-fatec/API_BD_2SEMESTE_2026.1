# 📘 User Story - US16

**Como professor, desejo que o calendário acadêmico esteja sempre atualizado mesmo nos próximos semestres.**  

<br>

## 📌 Explicação  

Essa funcionalidade permite que o calendário acadêmico seja atualizado a cada novo semestre, garantindo que os dias letivos, feriados e eventos acadêmicos estejam sempre corretos no planejamento das aulas.<br>  
O objetivo é evitar que o professor precise cadastrar manualmente um novo calendário a cada semestre, reduzindo retrabalho e inconsistências no cronograma.<br>  

<br>

## 📌 Informações exibidas  

O calendário acadêmico deve apresentar:<br>  

- Dias letivos<br>  
- Feriados<br>  
- Eventos acadêmicos<br>  
- Período do semestre<br>  

<br>

## 📌 Regras de Negócio  

- Deve ser possível atualizar o calendário acadêmico para novos semestres<br>  
- O calendário atualizado deve substituir o calendário anterior<br>  
- As alterações do calendário devem refletir automaticamente no planejamento das aulas<br>  
- Feriados e eventos acadêmicos devem ser considerados automaticamente após atualização<br>  
- O professor não deve precisar cadastrar manualmente os dias letivos de cada semestre<br>  

<br>

## 📌 Critérios de Aceitação  

- Deve ser possível utilizar um calendário acadêmico atualizado para um novo semestre<br>  
- O planejamento deve refletir corretamente o novo calendário<br>  
- Feriados e eventos acadêmicos devem aparecer corretamente após atualização<br>  
- O professor não deve precisar recriar manualmente todas as datas de aula<br>  

<br>
