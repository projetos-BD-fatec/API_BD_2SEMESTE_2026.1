# 📘 User Story - US02

Como professor, desejo visualizar os tópicos cadastrados, bem como suas quantidades de aulas mínimas e máximas para acompanhar a distribuição do conteúdo.
<br>

## 📌 Explicação

Essa funcionalidade permite que o professor visualize todos os tópicos (conteúdos) cadastrados para uma disciplina, e as quantidades mínimas e máximas de aulas para lecionar a disciplina.<br>
<br>

## 📌 Informações exibidas

Para cada tópico, devem ser apresentados:<br>

- Nome/descrição do tópico (ex: Introdução, Modelagem de Dados, SGDB)<br>
- Quantidade de aulas mínimas e máximas<br>
<br>

## 📌 Regras de Negócio
- Devem ser exibidos todos os tópicos cadastrados da disciplina selecionada<br>
- Cada tópico deve ter quantidade mínima e quantidade máxima de aulas <br>
<br>

## 📌 Critérios de Aceitação
- Ao acessar a disciplina, os tópicos cadastrados devem ser exibidos<br>
- Cada tópico deve ter quantidade mínima e quantidade máxima de aulas <br>
- A lista deve estar organizada e de fácil visualização (ex: tabela ou lista)<br>
<br>

## 📌 Exemplo
| Tópico             | Mínimo | Máximo |
| ------------------ | ---------------- | ----------------- |
| Apresentação       | 1                | 1                 |
| SGDB            | 3                | 5                 |
| Modelagem de Dados | 4                | 6                 |
