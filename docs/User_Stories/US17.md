# 📘 User Story - US17  

**Como professor, desejo editar o conteúdo de uma aula já planejada.**  

<br>

## 📌 Explicação  

Essa funcionalidade permite que o professor altere o tópico associado a uma aula já planejada.<br>  
O objetivo é facilitar ajustes no cronograma ao longo do semestre, permitindo reorganizar conteúdos sem a necessidade de recriar todo o planejamento.<br>  

<br>

## 📌 Informações exibidas  

Para cada aula planejada, devem ser apresentados:<br>  

- Data da aula<br>  
- Dia da semana<br>  
- Horário da aula<br>  
- Tópico atualmente associado<br>  
- Lista de tópicos disponíveis para substituição<br>  

<br>

## 📌 Regras de Negócio  

- Deve ser possível alterar o tópico de uma aula já planejada<br>  
- Apenas tópicos cadastrados para a disciplina podem ser selecionados<br>  
- A alteração deve atualizar automaticamente o planejamento<br>  
- A quantidade de aulas atribuídas a cada tópico deve ser recalculada após a edição<br>  
- Não deve ser permitido associar conteúdo a datas inválidas ou proibidas<br>  

<br>

## 📌 Critérios de Aceitação  

- Deve ser possível editar o tópico de uma aula planejada<br>  
- O novo tópico deve ser salvo corretamente<br>  
- O planejamento deve refletir a alteração realizada<br>  
- A contagem de aulas dos tópicos deve ser atualizada automaticamente<br>  
- Apenas tópicos válidos devem estar disponíveis para seleção<br>  

<br>
