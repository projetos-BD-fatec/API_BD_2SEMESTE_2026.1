# 📘 User Story - US08

Como professor, desejo ser avisado quando a carga horária não estiver sendo cumprida.

<br>

## 📌 Explicação

Essa funcionalidade permite alertar o professor quando o planejamento das aulas não atende à carga horária mínima da disciplina.<br>
O objetivo é evitar que o semestre seja planejado de forma incompleta.<br>

<br>

## 📌 Informações exibidas

Devem ser apresentados:<br>

- Aviso indicando que a carga horária não foi atingida<br>
<br>

## 📌 Regras de Negócio
- O alerta deve ser exibido quando as aulas planejadas forem menores que o total necessário<br>
- O alerta deve ser atualizado conforme o planejamento evolui<br>
- O alerta deve desaparecer quando a carga horária for atingida<br>
<br>
